# Where AI Fits in the IR Cycle

| IR Phase | AI Use Case |
|----------|-------------|
| **Preparation** | Threat modeling assistance, playbook drafting |
| **Detection** | Anomaly detection, alert correlation |
| **Triage** | Priority scoring, false-positive filtering |
| **Response** | Automated containment, IR report generation |
| **Post-incident** | Root cause summarization, timeline reconstruction |
